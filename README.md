# Implementación SIEM con Wazuh

## Descripción
Este proyecto consiste en la implementación de un entorno SIEM utilizando Wazuh, integrando dispositivos de red y seguridad para centralizar y analizar eventos.

## Tecnologías utilizadas
- Wazuh
- WatchGuard M690
- Switches Cisco
- Syslog
- Linux
- Visual Code Studio
- dashboards
- Telegram

## Funcionalidades
- Integración de firewall y switches
- Recepción de logs en tiempo real
- Creación de decoders personalizados
- Creación de reglas personalizadas
- Clasificación de eventos (tráfico, VPN, IDS)
- Reducción de falsos positivos

## Trabajo realizado
Durante este proyecto se desarrollaron decoders y reglas personalizadas para interpretar logs provenientes de dispositivos reales, permitiendo mejorar la visibilidad y clasificación de eventos de seguridad.
